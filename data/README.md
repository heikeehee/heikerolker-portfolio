# Data

## Raw data

Raw data is not committed to this repository.

### LSMS-ISA Tanzania (Projects 01, 02, 03)

Data source: World Bank LSMS-ISA Tanzania National Panel Survey Wave 4 (2014-15)
Access: https://microdata.worldbank.org/index.php/catalog/2862
Registration required: Yes
Licence: Public use, not for redistribution

After downloading, place files in: `data/raw/lsms/`

### Plate waste data (Project 04)

See `04-plate-waste-impact/data/README.md`

## Processed outputs

Processed outputs are generated by running the pipeline scripts in each project folder.
See each project's README.md for the correct execution order.

- `data/processed/01/` — outputs from 01-smallholder-material-flow
- `data/processed/02/` — outputs from 02-survey-harmonisation
- `data/processed/03/` — outputs from 03-food-system-segmentation

## Reproducing the analysis

1. Download LSMS-ISA files and place in `data/raw/lsms/`
2. Open the portfolio `.Rproj` file in RStudio
3. Run `source("scripts/packages.R")`
4. Follow execution order in the relevant project README
