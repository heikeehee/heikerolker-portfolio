# Heike Rolker — Data Portfolio

**Food Systems Data Analyst | PhD Researcher**

This portfolio brings together analytical work developed during doctoral research in food systems, translated into reproducible, industry-standard projects. The work spans smallholder agriculture in Tanzania, household food system analysis, and food waste measurement in the UK. Each project applies methods from the research — resource flow tracking, probabilistic modelling, survey harmonisation, segmentation — structured to meet the standards expected in a professional data environment.

Data is sourced from the World Bank LSMS-ISA programme (publicly available) and a UK plate waste study. Raw data is not stored in this repository — see each project's `data/README.md` for download instructions.

---

## Projects

| Folder | Description |
|---|---|
| [`01-smallholder-material-flow`](./01-smallholder-material-flow/) | Resource flow tracking from agricultural production to household consumption using Tanzania national survey data (LSMS-ISA), including probabilistic uncertainty modelling |
| [`02-survey-harmonisation`](./02-survey-harmonisation/) | Linking two survey instruments with different recall periods via bootstrapping to produce comparable consumption estimates at household level |
| [`03-food-system-segmentation`](./03-food-system-segmentation/) | Multivariate profiling and clustering of household food system typologies using combined production and consumption indicators |
| [`04-plate-waste-impact`](./04-plate-waste-impact/) | Quantifying food waste by meal component and translating losses into nutrient and carbon footprint equivalents |
